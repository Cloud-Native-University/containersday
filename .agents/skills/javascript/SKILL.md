# JavaScript & Libraries Skill

This skill focuses on adding interactivity and dynamic behavior to web applications using vanilla JavaScript and popular libraries.

## Core JavaScript (ES6+)
- **Async/Await & Fetch**: For data retrieval and asynchronous logic.
- **Destructuring & Spread**: Clean object/array manipulation.
- **Modules**: `import` and `export` for modular code.
- **Classes**: Structured object-oriented programming when appropriate.

## DOM Interaction
- **Event Delegation**: Handling events efficiently on parent elements.
- **Intersection Observer**: For lazy loading and scroll-triggered animations.
- **Mutation Observer**: Tracking changes in the DOM.

## Libraries & Integration
- **Animations**: GSAP (GreenSock), Framer Motion, or simple CSS-linked logic.
- **UI Components**: Alpine.js or HTMX for lightweight reactivity.
- **Data Visualization**: Chart.js or D3.js.
- **Utilities**: Lodash, date-fns, or small specialized micro-libraries.

## Performance
- **Debouncing & Throttling**: Limiting execution of high-frequency events (resize, scroll).
- **Code Splitting**: Loading only necessary scripts.
- **Efficient Loops**: Preferring `map`, `filter`, `reduce` for clarity and performance.

## Best Practices
1. **Unobtrusive JavaScript**: Ensure the page is functional even if JS fails to load.
2. **Memory Management**: Clean up event listeners and intervals to prevent leaks.
3. **Vanilla First**: Only use a library if it provides significant value over vanilla JS.
4. **Tooling**: Familiarity with Vite, Webpack, or Hugo's built-in ESBuild integration.
